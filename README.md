# 吃吃管家Line Bot

## 使用測試

請先加入好友

QRCODE

![alt tag](https://iili.io/KIXAvf9.md.jpg)

或是手機直接點選 [https://lin.ee/4NgW1BU](https://lin.ee/4NgW1BU)

## 主要功能：塔羅牌詳細占卜

吃吃請幫我占卜，我想問XXX嗎？

例：吃吃請幫我占卜，我今天考試會順利嗎？
![K7FPFun.png](https://iili.io/K7FPFun.png)

## 其他功能：
### 群聊

吃吃管家在群聊也可以自由回應任何成員，也會自主選擇沉默(以輸出指令的形式)
![K7KTQHv.png](https://iili.io/K7KTQHv.png)
